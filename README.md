# Pranav Finance Dashboard

Personal finance dashboard — Nov 2024 to present.

Live at: https://athristt.github.io/personal_finance-dashboard/

## How to update monthly
1. Get new bank statements (NIC Asia, Nabil, Prime Bank, eSewa, Khalti)
2. Send them to Claude in the same chat with the message: `/finance-new-month-dashboard`
3. Download the new `index.html` and replace this file
4. Push to GitHub — link updates automatically

## Accounts tracked
- NIC Asia Bank
- Nabil Bank  
- Prime Commercial Bank
- eSewa (wallet)
- Khalti (wallet)
